DigFinder Web Services API



Craigslist Housing search request:

JSON Example: http://localhost:8080/digfinderserver/search/area=sby&nh=santa clara&bedrooms=4

Default output format is JSON. A request for XML output has the following form:

XML Example: http://localhost:8080/digfinderserver/search/xml/area=sby&nh=santa clara&bedrooms=4


Craigslist Housing neighborhood list request:

JSON Example: http://localhost:8080/digfinderserver/nhoods

XML Example: http://localhost:8080/digfinderserver/xml/nhoods
